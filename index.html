Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: "stages" or is a record of what was changed *must do this after any chages so it records to git*
  • git commit -m: Note or "message" to let users know what you changed
  • git pull origin main: This ensures that we have the latest commits
  • git push origin feature/starter-code: Will push code to my branch *DO NOT push to main branch!!*
  • git init: Intializes a new Git repo and begins tracking an existing directory. It adds a hidden subfolder within the existing directory that houses the internal data structure required for version control
  • git clone: Creates a local copy of a project that already exists remotely. The clone includes all the project's files, history and branches
  • git branch: Showes branches being worked on
  • git help "command": Will open up a detailed help page about the chosen command
